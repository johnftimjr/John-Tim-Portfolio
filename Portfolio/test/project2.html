Project Title: Ice Hockey Stick Design

Tools Used: CAD Software

Project Overview: This project involved designing a custom ice hockey stick using CAD software. The focus was on optimizing materials and enhancing ergonomics based on input from NCAA athletes, followed by rigorous testing to ensure durability and performance.

Project Objectives

  • Design a durable, high-performance ice hockey stick.
  • Use ergonomic data to improve stick design.
  • Test the final prototype in real-world conditions.

Project Outcomes

The final design reduced weight by 20% while maintaining structural integrity. Athlete feedback was overwhelmingly positive, highlighting improved grip and control. Plans for future enhancements include experimenting with composite materials to further reduce weight.
